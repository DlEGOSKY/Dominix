# RELIC_SYSTEM.md — Dominix

## Objetivo
Definir el sistema de reliquias y mejoras que convierte una partida en una build.

## Qué es una reliquia
Una reliquia es un modificador pasivo o condicional que altera:
- score
- patrones
- comportamiento de ciertas fichas
- valor de ciertos números
- recompensas de ronda

## Reglas
- deben ser fáciles de leer
- deben cambiar cómo juegas
- no deben depender de texto excesivo para entenderse
- deben empujar estilos de run distintos

## Tipos de reliquia

### A. Reliquias de número
Ejemplos:
- Eco Par: los números pares reciben bonus
- Pulso Bajo: 0 a 3 ganan valor extra
- Corona Alta: 4 a 6 ganan multiplicador

### B. Reliquias de patrón
Ejemplos:
- Doble Corona: los dobles multiplican más
- Final Pesado: el cierre exacto vale triple
- Cadena Tensa: cadenas largas suman más

### C. Reliquias de estructura
Ejemplos:
- Mano Extendida: +1 ficha en mano
- Aire Extra: +1 descarte
- Taller Negro: permite mutar una ficha gratis tras cierto evento

### D. Reliquias de riesgo
Ejemplos:
- Todo o Nada: si no activas patrón, pierdes valor; si activas uno raro, explota el score
- Borde Vivo: los extremos importan más, pero castigan errores

## Recompensas entre rondas
Opciones posibles:
- elegir una de 3 reliquias
- eliminar una ficha débil del set
- duplicar una ficha poderosa
- convertir un número en otro
- reforzar score base de una categoría

## Regla del MVP
El MVP no necesita 50 reliquias.
Necesita 10 a 15 reliquias buenas que cambien de verdad las decisiones.
